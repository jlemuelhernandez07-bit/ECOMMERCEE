# Nike Store PH - E-commerce Website

A fully functional Nike shoe store for the Philippines market.

## Features
- User authentication (login/register)
- Product catalog with filtering
- Shopping cart with quantity management
- Multiple payment options (GCash, Credit Card, COD)
- Responsive design for all devices
- Product details modal
- Customer testimonials

## Live Demo
[View on GitHub Pages](https://jlemuelhernandez07-bit.github.io/nike-store-ph/)

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage for data persistence
- Font Awesome icons
